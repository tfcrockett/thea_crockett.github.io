Grab and Go Meals

A website allowing students in public schools locate the nearest school cafeterias that offer free meals. With the closure of schools and limited means of transportation due to Covid, students needing meals were stranded at home.A web page that captures the user location was crucial then for these individuals to get food on time.
Contribution - data collection, geocode addresses, convert information to GeoJSON, developed web page, created map icon.
Tools used - Excel, Javascript, geoJson, geocoding, leaflet, HTML, CSS, photoshop and Github.

Dating App: Machine Learning Model

Data from a SPEED DATING experiment was used to train and test a model that matches participants based on their skills and likeability.
Contribution - trained and tested the model with SVD, KNN and Matrix Factorization to determine matches between participants using a coefficient on their interests. -Copied the Matrix factorization coefficients from python and recreated it in Tableau for better visual representation and to predict how the users rate skills and likeability to generate a recommendation based on the prediction.
Tools used - Python, Flask, Javascript, Leaflet, API calls, HTML, CSS, Github.
